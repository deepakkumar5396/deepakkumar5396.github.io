# Deepak Kumar Portfolio

A modern one-page portfolio for Deepak Kumar, designed to feel polished, recruiter-friendly, and ready for quick deployment.

## Stack

- HTML
- CSS
- Vanilla JavaScript

## Why this stack

- No dependency installation needed
- Fast to load and easy to customize
- Simple to deploy on GitHub Pages, Netlify, or Vercel

## Local preview

Open `index.html` directly in a browser, or run:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Files

- `index.html` - page structure and content
- `styles.css` - design system, layout, motion, responsive styling
- `script.js` - reveal animations, stat counters, footer year
- `resume.html` - SEO-friendly web resume page
- `resume.css` - styling for the web resume page
- `generate_resume.py` - generates the downloadable PDF resume
- `Deepak_Kumar_Resume.pdf` - downloadable resume linked from the UI

## Regenerate resume PDF

```bash
python3 generate_resume.py
```
